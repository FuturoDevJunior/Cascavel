Quantum Security Framework

Developed for Red Teams and advanced pentesting

Cascavel is a modular and extensible offensive security framework designed to automate high-complexity pentesting workflows. With plugin-based architecture and native integrations, it enhances productivity and clarity in security assessments.

Features

  • 🔌 Plugin-based architecture
  • ⚙️ Integration with Subfinder, Amass, Nmap, Naabu, etc.
  • 📁 Built-in wordlists and vulnerability templates
  • 📝 Automatic Markdown reporting
